# <a name="data-catalog-developer-samples"></a>Andmete kataloogi arendaja näidised
Andmekataloogi minirakenduste arendamine abil andmete kataloogi REST API-ga. Andmete kataloogi REST API on ülejäänud vastavalt API, mis pakub Programmeerimisjuurdepääs andmekataloogi ressursse registreerida, märkmed ja otsige andmete varad programmiliselt.

Siin on meil github näidised.

- [Azure'i andmekataloogi kasutamise alustamine](https://azure.microsoft.com/documentation/samples/data-catalog-dotnet-get-started/)

  Valimi mis näitab, kuidas registreerida, otsida ja kustutada andmete varade, kasutades andmete kataloogi REST API Azure AD autentimiseks.

- [Hulgi registreerida ja märkmed](https://azure.microsoft.com/documentation/samples/data-catalog-dotnet-excel-register-data-assets/)

  Näide, mis näitab, kuidas hulgiredigeerimiseks register andmete varad Exceli töövihiku andmete kataloogi REST API ja Open XML-i abil.

- [Import/eksport tööriist](https://azure.microsoft.com/documentation/samples/data-catalog-dotnet-import-export/)

  Näide, mis näitab, kuidas kasutada andmete kataloogi REST API tuua Azure'i andmekataloogi varad ja serialiseerida need faili. See näitab ka kuidas võtta kogumi nimega JSON seeriasertide varad ja vajutage neid kataloogi. See toetab eksportimise alamhulga kataloogi otsimine päringu abil.
